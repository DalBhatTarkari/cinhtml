Syntax Error

These occur when the code violates the rules of the C language and are detected during compilation.

Code:

#include
int main()
{
    printf("Hello, world")
}

Runtime Error

These occur when the program is executed. The program may compile successfully but crash or behave unexpectedly at runtime.

Code:

#include
int main()
{
    int x;
    x=5/0;
}

Output:




Logical Error

The program runs without crashing, but the output is incorrect due to flawed logic due to the programmer.

Code:

#include
int main()
{
    int a=10,b=5;
    printf("Sum is %d",a-b);
}

Output:




Semantics Error

When a statement is syntactically correct but has no meaning, semantic errors occur. This is similar to grammatical errors.

Code:

#include
void main()
{
    int a=10, b=10, sum=0;
    a+b=sum;
}

Output:

if (strcmp(week , "SATURDAY") == 0 )

if (strcmp(week , "SATURDAY") != 0)
